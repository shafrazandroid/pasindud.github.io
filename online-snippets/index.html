Online-snippets

Dec 25th, 2012 | Comments

Currently I am developing a online snippet safe form where you can type code online Zor maybe copy and save it online, the code editor is online is fullscreen therefore there is distractions.I be using the laravel framework to do this project which is a really great framework and the Eloquent ORM with it’s relationship functionalities makes life easy to the programmers to work with.

Features

  • Login
  • Profile
  • Public Codes
  • Private Codes
  • Online Code Editor
  • Quick save without needing to login

Hope fully I will be getting a chance to learn laravel more and more.
